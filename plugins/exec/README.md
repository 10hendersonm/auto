# Exec Plugin

Tap into select hooks and run a command on the terminal

## Installation

This plugin is not included with the `auto` CLI installed via NPM. To install:

```sh
npm i --save-dev @auto-it/exec
# or
yarn add -D @auto-it/exec
```

## Usage

```json
{
  "plugins": [
    ["exec"]
    // other plugins
  ]
}
```
