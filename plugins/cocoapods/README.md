# Cocoapods Plugin

Use auto to version your cocoapod

## Installation

This plugin is not included with the `auto` CLI installed via NPM. To install:

```sh
npm i --save-dev @auto-it/cocoapods
# or
yarn add -D @auto-it/cocoapods
```

## Usage

```json
{
  "plugins": [
    ["cocoapods"]
    // other plugins
  ]
}
```
